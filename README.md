# Canil-nodets
Projeto Canil feito com Node e Typescript


### Pré-requisitos globais:

`npm i -g nodemon typescript ts-node`

#### Caso não tenha essas bibliotecas instaladas globalmente em seu computador
`npm i -g nodemon typescript ts-node`

### Instalação
`npm install`

### Para rodar o projeto
`npm run dev`
